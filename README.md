# Installation

## 1. Install gcc4mbed library

- Visit https://github.com/adamgreen/gcc4mbed 
- Install Linux version and set GCC4MBED_DIR, ROS_LIB_DIR

    ```sh
    export GCC4MBED_DIR=~/Project/07_Mbed_ros/gcc4mbed/adamgreen-gcc4mbed-0e9c0fb
    export ROS_LIB_DIR=~/ros/lib/ros_lib
    ```
- Use `make` command to build project
